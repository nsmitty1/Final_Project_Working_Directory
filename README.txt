Title:   Low levels of outdoor recreation alter wildlife behavior

# Authors: Mira L. T. Sytsma, Tania Lewis, Beth Gardner and Laura R. Prugh

# Journal: People and Nature
#-------------------------------------------------------------------------------------------------------#

The content of the Zip file:

- The file of "Remote camera data": data from all 40 remote cameras including location, date, time, species detected in each photo, number of individuals detected in each photo, and trigger mode (timelapse vs motion detection)

- The file of "Camera stations": locations, set up dates, take down dates and timeframes when cameras were not operational
